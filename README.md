# Система управления задачами с приоритетами

Командный проект по дисциплине «Основы программирования».

## Описание

Консольная программа для ведения списка задач. Для каждой задачи хранятся:

- название;
- приоритет (1–5);
- время выполнения (минуты);
- дедлайн.

## Реализованные структуры данных

| Структура | Назначение | Файл |
|-----------|------------|------|
| Бинарное дерево поиска | Сортировка и быстрый поиск по дедлайнам | `src/structures/binary_search_tree.py` |
| Стек | Отмена последних действий (добавление, удаление, изменение) | `src/structures/stack.py` |
| Очередь | Очередь задач на исполнение | `src/structures/queue.py` |

## Возможности

- добавление, удаление и изменение задач;
- вывод задач в порядке возрастания дедлайна (рекурсивный обход BST);
- быстрый поиск задачи с самым ранним и самым поздним дедлайном;
- очередь задач на исполнение;
- отмена последнего действия через стек.

## Запуск

```bash
python main.py
```

Требуется Python 3.10+.

## Формат дедлайна

При вводе дедлайна используйте формат: `ДД.ММ.ГГГГ ЧЧ:ММ`  
Пример: `15.06.2026 18:00`
